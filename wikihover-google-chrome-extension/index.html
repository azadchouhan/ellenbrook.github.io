Wikihover is a Google Chrome extension that completely removes the most annoying aspect of browsing Wikipedia; getting lost in a sea of tabs. That's right, this Chrome extension immediately makes browsing Wikipedia exponentially more enjoyable for just about anybody who installs it.

WikiHover is open source and everything is freely available on the Wikihover Github repo. WikiHover is available in the Chrome web store for free. All that I ask if that if you like it, rate it and share it!
